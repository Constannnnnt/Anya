# Code of Conduct

## Our Standard

We are committed to a respectful, inclusive, and harassment-free community.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and technical outcomes.
- Assume good intent and ask clarifying questions before escalating.

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks.
- Doxxing or disclosure of private information.
- Repeated bad-faith disruption of collaboration.

## Enforcement

Report issues to repository maintainers through private GitHub channels.
